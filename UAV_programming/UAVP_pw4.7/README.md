# Практическая работа 4.7: Симулятор квадрокоптера

## Описание
Реализация математической модели мультироторного БЛА (квадрокоптер) с каскадной системой управления на основе ПИД-регуляторов.

### Особенности:
- Полная динамика 6 DOF (Newton-Euler уравнения)
- Каскадная система управления (4 контура): Позиция → Скорость → Угол → Угловая скорость
- Учёт физических эффектов: сила Кориолиса, гироскопический эффект, аэродинамическое сопротивление
- Интегрирование методом Рунге-Кутта 4-го порядка (RK4)
- Следование маршруту из массива точек (waypoints)

## Требования

- Python 3.7+
- Библиотеки: `numpy`, `matplotlib`
