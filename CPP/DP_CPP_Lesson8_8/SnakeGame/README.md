# 🐍 Игра «Змейка» на Qt5

Практическая работа по фреймворку Qt-5.

## 📋 Требования

- Компилятор с поддержкой C++17 (GCC 9+, Clang 10+, MSVC 2019+)
- Qt 5.12 или новее (модули: Core, Gui, Widgets)
- CMake 3.14+

## 🚀 Сборка и запуск

### Linux / macOS
```bash
mkdir build && cd build
cmake ..
cmake --build .
